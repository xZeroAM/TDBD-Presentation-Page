UrbanGym

Sistema de Gestión de Usuarios y Entrenadores para UrbanGym

El proyecto tiene como objetivo desarrollar un sistema de información que permita a UrbanGym gestionar de manera eficiente a sus usuarios y entrenadores, así como ofrecer rutinas personalizadas a los usuarios a través de la página web.

Objetivos:

  • Crear una base de datos centralizada para almacenar información de los usuarios, entrenadores y membresías.

  • Permitir a UrbanGym administrar los datos de los usuarios, incluyendo su información personal, membresías y registros de asistencia.

  • Facilitar la gestión de los entrenadores, permitiendo asignar clientes a cada entrenador y llevar un seguimiento de las rutinas personalizadas.

  • Desarrollar una interfaz web intuitiva para que los usuarios puedan acceder a su información, consultar su progreso, recibir rutinas personalizadas y reservar clases.

  • Mejorar la eficiencia operativa de UrbanGym al automatizar tareas administrativas y reducir la carga de trabajo manual.
